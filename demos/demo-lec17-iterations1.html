apply() family functions

  • apply() : apply a function on an R array (e.g. 2-dim, N-dim)

  • lapply() : apply a function on an R list

  • sweep() : sweep elements of a matrix

Some useRs refer to these functions as vectorized loop functions

apply(X, MARGIN = 2, FUN = mean)
[1] 0.6418399 0.3698680 0.5441375
apply(X, MARGIN = 2, FUN = median)
[1] 0.7243046 0.3896737 0.5143078
apply(X, MARGIN = 2, FUN = min)
[1] 0.166371785 0.001136587 0.226467179

What about the range: max - min?

range(X[, 1])
[1] 0.1663718 0.9897369
myrange = function(x) {
  max(x) - min(x)
}

myrange(X[,1])
[1] 0.8233652
# we can use our own functions
apply(X, 2, myrange)
[1] 0.8233652 0.9505222 0.7389481
# we can also use "anonymous" functions (i.e. lambda)
apply(X, 2, function(y) max(y) - min(y))
[1] 0.8233652 0.9505222 0.7389481
# we can also use "anonymous" functions (i.e. lambda)
apply(X, 2, \(y) max(y) - min(y))
[1] 0.8233652 0.9505222 0.7389481
