Lec-17: Iterations (part 1)

Published

October 7, 2024

Motivation

Let’s consider a toy matrix of random numbers arrange in 10 rows and 3 columns:

set.seed(1234) # for reproducibility
x = matrix(runif(30), 10, 3)
x
             [,1]      [,2]       [,3]
 [1,] 0.113703411 0.6935913 0.31661245
 [2,] 0.622299405 0.5449748 0.30269337
 [3,] 0.609274733 0.2827336 0.15904600
 [4,] 0.623379442 0.9234335 0.03999592
 [5,] 0.860915384 0.2923158 0.21879954
 [6,] 0.640310605 0.8372956 0.81059855
 [7,] 0.009495756 0.2862233 0.52569755
 [8,] 0.232550506 0.2668208 0.91465817
 [9,] 0.666083758 0.1867228 0.83134505
[10,] 0.514251141 0.2322259 0.04577026

Goal: say we want to obtain summary statistics for each of the columns, for example the means (or medians, minimima, range, etc).

Naive Approach

To get the column means, conceptually speaking, we need to traverse every column and get the mean:

xmeans = c(mean(x[ ,1]), mean(x[ ,2]), mean(x[ ,3]))
xmeans
[1] 0.4892264 0.4546337 0.4165217

This is too repetitive, and not very efficient.

But the important step can be generalized in an abstract sense as:

mean(x[ ,pos])

where pos indicates position ranging from the first column to the third column.

For-loop

Here’s how to compute the column means with a for loop:

# initialize output vector
xmeans = rep(0, 3)

# iterative steps
for (pos in 1:3) {
  xmeans[pos] = mean(x[ ,pos])
}

xmeans
[1] 0.4892264 0.4546337 0.4165217

And we can make it even more general to any number of columns in x:

# initialize output vector
xmeans = rep(0, ncol(x))

# iterative steps
for (pos in 1:ncol(x)) {
  xmeans[pos] = mean(x[ ,pos])
}

xmeans
[1] 0.4892264 0.4546337 0.4165217

Generic blue print of a for loop

for (iterator in vector) {
  # repetitive steps
}

A for loop takes three ingredients:

  • iterator: auxiliary object

  • in keyword

  • vector: a vector whose length will determine the number of iterations

Apply

Alternative to for loops, we also have the apply() function which can be used for arrays, to apply a function to all the rows or columns.

Here’s how to use apply() to get the column means, medians, and range:

xmeans = apply(x, 2, mean)
xmeans
[1] 0.4892264 0.4546337 0.4165217
xmedians = apply(x, 2, median)
xmedians
[1] 0.6157871 0.2892696 0.3096529
# range: max - min
# anonymous
apply(x, 2, function(u) {max(u)-min(u)})
[1] 0.8514196 0.7367107 0.8746622
# function defined outside apply
myrange = function(u) {
  max(u)-min(u)
}
apply(x, 2, myrange)
[1] 0.8514196 0.7367107 0.8746622

What if the input matrix has missing values?

# copy of x
y = x
y[1,1] = NA

mean(y[,1], na.rm = TRUE)
[1] 0.5309512
apply(y, 2, mean, na.rm = TRUE)
[1] 0.5309512 0.4546337 0.4165217

While Loop

xmeans = rep(0, ncol(X))

i = 1

while (i <= ncol(X)) {
  xmeans[i] = mean(X[,i])
  i = i + 1
}
