Example using Abbreviated classes

  • \\s : space; \\S not spaces

  • \\w : “word” character (e.g. upper/lower letters, digits, _); \\W not a word

    i.e. [a-zA-Z0-9_]

  • \\d : digits (ie. [0-9] , [:digit:] ); \\D not a digits

  • \\b : word boundaries; \\B not a word boundary
