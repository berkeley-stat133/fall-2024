Vectors (last time)

Vectors and their data types (data primitives)

  • logical (boolean)
  • integer
  • double or real (float)
  • character (string)

Herer are some simple examples:

# logical
a = c(TRUE, FALSE, NA)

# integer
b = c(2L, 4L, 6L)

# double (real)
c = c(1, 2, 3, 4)
d = c(1.1, 2.2, 3.3, 4.0)

# character
e = c("a", "b", "c", "d")

Question from last time

As you know, we can use names() to assign names to the elements of a vector

# hypothetical example
names(c) = e

Question: Do the input vector c and the vector of names e must be of the same length? Answer: Yes, and No. It depends. Let’s see three scenarios:

# this works
x = c(2, 4, 6)
abc = c("a", "b", "c")

names(x) = abc
x
a b c 
2 4 6 
# this fails
y = c(2, 4, 6)
abcd = c("a", "b", "c", "d")

names(y) = abcd
Error in names(y) = abcd: 'names' attribute [4] must be the same length as the vector [3]
# this works (but you get a missing named element)
z = c(2, 4, 6, 8)

names(z) = abc
z
   a    b    c <NA> 
   2    4    6    8

Coercion

What R does when it encounters an operation/computation that deals with different data types.

Implicit Coercion

# implicit coercion rules
int = c(2L, 4L, 6L)
logi = c(TRUE, FALSE)

vec = c(FALSE, 1L)
vec
[1] 0 1

Remember the hierarchy of data types:

logical < integer < double < character

Vectors are Atomic objects.

Every element of a vector has to be of the same data type

Explicit Coercion Functions

as.logical(c(0, 1, 2, 133))
[1] FALSE  TRUE  TRUE  TRUE
as.integer(c(1, 2.2, 3.33, 4.444))
[1] 1 2 3 4
as.double(c("a", "b", TRUE, 133))
Warning: NAs introduced by coercion
[1]  NA  NA  NA 133

Recycling

When you operate with 2 vectors of different length, the elements of the shorter vector are recycled as many as times as necessary to match the length of the longer vector.

x = c(2, 4, 6, 8)
x
[1] 2 4 6 8
y = c(2, 1)
y
[1] 2 1
x + y
[1] 4 5 8 9
c(2, 4, 6, 8) + c(2, 1, 2, 1)
[1] 4 5 8 9
u = c(2, 4, 6, 8, 10)
u
[1]  2  4  6  8 10
v = c(2, 1)
v
[1] 2 1
u + v
Warning in u + v: longer object length is not a multiple of shorter object
length
[1]  4  5  8  9 12
